Project Overview

Purpose and Goals

Optimizing ordering and dining experiences by modernizing the canteen with technology. Key objectives:

  • Reduce wait times & streamline queues.
  • Implement a fully cashless payment system.
  • Enable online balance checks and canteen status.
  • Improve staff efficiency and reporting abilities.

Primary Users

Designed for multiple environments like:

  • University Canteens
  • Corporate Cafeterias
  • Controlled Food Services

Hardware Components

Our system integrates reliable hardware components for seamless operations including:

  • ESP32 Board: Core microcontroller for control and processing.
  • Raspberry Pi Camera: Visual monitoring, crowd detection.
  • RFID Writer & Reader: Contactless user authentication and transactions.
  • Fingerprint Sensor: Secure biometric authentication.
  • Keypad Matrix: Input interface for user interactions.
  • LCD Display: Live informational display accessible to users.
